# NVM (Node.js Version Manager)

> NVM
> <br> https://github.com/nvm-sh/nvm

### Curl을 사용하여 설치 (0.39.7 버전 기준):

```zsh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

### `~/.zshrc` 적용:

```zsh
source ~/.zshrc
```

### Node.js LTS 버전 설치:

```zsh
nvm i —lts
```

### 설치된 Node.js 버전 중 원하는 버전으로 변경:

```zsh
nvm use “[버전]”
```

### npm 업데이트:

```zsh
npm i -g npm
```

### yarn 설치:

```zsh
npm i -g yarn
```

### yarn 버전을 2로 초기화:

```zsh
yarn init -2
```
